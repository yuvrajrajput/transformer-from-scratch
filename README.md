# Transformer From Scratch

Research-oriented implementations of neural architectures using PyTorch.

## Implemented Models

### 1. Small Language Model (SLM)
- English to French translation
- Sequence modeling
- Attention-based architecture
- Token embedding experiments

### 2. Natural Language Autoencoder
- Latent representation learning
- Text reconstruction
- Encoder-decoder architecture
- Embedding compression experiments

## Tech Stack
- Python
- PyTorch
- Google Colab
- NumPy
- Matplotlib

## Goals
This repository is focused on understanding modern NLP architectures from first principles by implementing models manually rather than relying heavily on high-level frameworks.
